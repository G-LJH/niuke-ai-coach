# 牛客AI面试教练 - AI Agent 约束总则

## 项目说明
基于阿里百炼模型的面试辅导工具，包含：
- 面经抓取与分析
- 简历深度解析
- 模拟面试题目预测

## 核心约束（AI 必须遵守）
1. **先定义 Schema**：任何工具函数的输入输出必须先在 `harness/schemas.md` 中定义。
2. **网络请求**：只允许访问 `nowcoder.com` 和阿里百炼 API 域名。
3. **密钥保护**：所有敏感信息从环境变量读取，不得硬编码。
4. **爬虫限速**：牛客网请求间隔 ≥ 1 秒。
5. **断点续传**：长流程任务必须按 `harness/state.md` 记录进度。

## 目录约束
- `docs/`：功能流程说明
- `harness/`：工程约束定义
- 业务代码目录由 AI 自行组织

## 禁止事项
- 不得绕过安全规则直接操作文件或网络
- 不得在代码中写死 API 密钥
- 不得跳过 Schema 定义直接写函数
